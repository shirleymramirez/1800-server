# 1800-server

Instructions:
1. Make sure to install node version 18.18.2
2. Git clone the repo
3. Run 'npm install'
4. Run 'node index.js'
5. Open browser and go to http://localhost:8081/.

7. Go to http://localhost:8081/posts to see an array of object posts.
